# Communication Systems from Scratch

1. **bgp** — BGP finite state machine + path selection
2. **tcp_ip_stack** — User-space networking (TUN → IPv4 → ICMP → UDP → TCP)
3. **tcp_chat** — Secure chat (TCP → TLS → E2EE with RSA + AES-GCM)
4. **audio_modem** — FSK modulation, FFT, CRC — TCP packets as sound through air
5. **quic** — QUIC from scratch (UDP → reliability → DH → 0-RTT → migration → frames)
6. **bbr** — Google's BBR congestion control
7. **protobuf** — Protocol Buffers wire format (varints → tags → encode/decode)
8. **http3** — HTTP/3 on top of QUIC (frames → headers → working client/server)
